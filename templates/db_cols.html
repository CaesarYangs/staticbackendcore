How to write queries
  • Use a JSON syntax
  • A clause is an array ["field", "operator", "value"]

  • Wrap all your clauses into a parent array: [["field", "=", "value"], ["field2"...]]
  • Available operators: =, !=, <, > <=, >=, in, !in
